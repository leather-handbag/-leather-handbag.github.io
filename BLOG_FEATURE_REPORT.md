# 博客与评论功能报告

博客与评论已经迁移到 Supabase 数据层：

- 私有文章仅作者和管理员/站长可读，公开文章对访客开放。
- 文章作者只能修改自己的文章；评论作者只能修改自己的评论。
- 管理员只能删除普通用户内容，不能修改作者内容，也不能处理站长或其他管理员。
- Markdown、LaTeX、安全链接和 HTML 转义策略保持不变。
- 前端敏感内容提示之外，数据库触发器还会执行最终审核、删除和封禁。
- 文章、评论和工作站留言会显示账号资料、等级颜色以及管理员/站长标签。

数据库表、RLS、审核触发器和部署操作见 [SUPABASE_SETUP_REPORT.md](SUPABASE_SETUP_REPORT.md)。旧版 `localStorage` 博客不会自动上传，避免在用户不知情时公开本地草稿；需要迁移时应先人工检查文章可见性和敏感内容。
